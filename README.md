# Sentinel_landscape_delin
Délimitation des ZM a l'échelle du territoire

- RPG_filt_grid_creation a pour objectif de découper le territoire en subdivisions pour alléger/permettre les calculs à l'échelle des pixels.
Ce script aboutit à un ensemble de grilles identifiées par "Gridfilt" + un numéro de ligne et de colonne
- RPG_overgrid séléctionne les parcelles contenant au minimum 3 années des cultures de réference sur la période 2016 - 2020 et crée pour ces parcelles un identifiant commun.
